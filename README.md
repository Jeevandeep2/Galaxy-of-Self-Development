# 🌌 Galaxy of Self Development

A mind-blowing, immersive portfolio website featuring a realistic 3D universe background, magnetic cursor effects, and stunning animations.

## 🚀 Features

- **Realistic 3D Universe**: Three.js powered galaxy with 8000+ particles, nebula clouds, and floating geometric shapes
- **Magnetic Cursor System**: Multi-layer cursor with hover effects and magnetic attraction
- **GSAP Animations**: Smooth scroll-triggered animations and parallax effects
- **Responsive Design**: Mobile-first approach with breakpoints for all devices
- **30 Project Showcase**: Complete grid of self-development projects
- **Interactive Skills Constellation**: SVG-based animated skill network

## 🎯 Vision Statement

"Transforming self-development into an immersive cosmic journey where every project is a star in your personal galaxy of growth."*

This isn't just a portfolio—it's an **interactive 3D universe** showcasing 30+ transformative self-development projects. Built with cutting-edge web technologies, it delivers a mind-bending experience that combines stunning visuals with purposeful content.



## ✨ Cosmic Features


### 🌌 **Realistic 3D Universe**
- **8000+ Animated Particles** forming a living galaxy
- **Nebula Cloud Systems** with dynamic shaders
- **Floating Geometric Shapes** that respond to mouse movement
- **Real-time Lighting** with WebGL acceleration
- **60FPS Performance** optimized rendering



### 🧲 **Magnetic Cursor System**
- **Multi-layer Cursor Trail** with physics-based animation
- **Magnetic Hover Effects** on all interactive elements
- **Smooth GSAP Transitions** throughout the interface
- **Custom Cursor States** for different interactions
- **Touch-Optimized** for mobile devices



### 🎭 **GSAP Animations**
- **Scroll-Triggered Animations** with parallax effects
- **Staggered Reveals** for project cards
- **Smooth Page Transitions** between sections
- **Timeline Sequences** for complex animations
- **Performance Optimized** with will-change properties

### 📱 **Responsive Design**
- **Mobile-First Architecture** with progressive enhancement
- **Adaptive Particle Count** (mobile: 2000, desktop: 8000)
- **Flexible Grid System** using CSS Grid and Flexbox
- **Cross-Browser Compatibility** (Chrome, Firefox, Safari, Edge)
- **Progressive Web App** ready with service worker


### 🎯 **30 Project Showcase**
- **6 Categories**: Career, Mindset, Finance, Relationships, Health, Learning
- **Interactive Filtering System** with smooth transitions
- **Animated Project Cards** with hover effects
- **Live Demo Links** to deployed projects
- **GitHub Integration** for source code access


### 🌐 **Interactive Skills Constellation**
- **SVG-Based Animated Network** of skills
- **Dynamic Connection Lines** between related technologies
- **Hover Effects** revealing skill details
- **Animated Drawing** of constellation paths
- **Responsive Scaling** for all screen sizes
